وێبسایتێکی تەواوی فیلم و زنجیرە و مانگا و ئەنیمیم بۆ دروستبکە لە شێوەی kurdsubtitle.net بە ئەم تەکنەلۆژیا و تایبەتمەندیانەی خوارەوە:

## تەکنەلۆژیای بەکارهاتوو:
- **ڕوکار**: Next.js 14+ (App Router)
- **کۆمپۆنێنتەکان**: Mantine UI
- **ستایل**: TailwindCSS
- **بەکئێند**: Node.js لەگەڵ Express.js
- **داتابەیس**: MongoDB لەگەڵ Mongoose
- **سێرڤەری وێنە**: سێرڤەرێکی جیای Express.js لەگەڵ Sharp.js و Redis بۆ خاوێنکردنەوەی داینامیکی وێنە
- **زمان**: کوردی (سۆرانی) بۆ هەموو دەقی وێبسایتەکە

## مەرجە گرنگەکان:
1. **هیچ سێرڤەرێکی ناوخۆیی بۆ ڤیدیۆ نەبێت** - تەنها لینکی سێرڤەری دەرەکی کە ئەدمین زیادی دەکات (وەک vidmoly و هتد)
2. **هیچ تایبەتمەندیەکی پلەیەری ڤیدیۆ نەبێت** (بێ: پێشخستن، وەستان، گەڕان، گەڕانەوە) - تەنها نیشاندانی پلەیەری دەرەکی
3. پڕۆژەکە دەبێت لە 100 فایل زیاتر بێت
4. هەموو کۆدەکان بێ هەڵە و ئامادەی بەکارهێنان بن
5. فایلێکی جیا بۆ ڕێکخستنی لۆگۆ و فۆنتەکان دروستبکە

## بەشە سەرەکییەکان (دەبێت بە تەواوی جیا بن):
1. **فیلم** (تاک فیلمەکان)
2. **زنجیرە فیلم** (فرانچایزەکان وەک Avatar، Fast & Furious)
3. **زنجیرە** (زنجیرە تەلەفزیۆنییەکان)
4. **٢٥٠ باشترینی مێژوو**
5. **هەواڵ**
6. **ژیانامە**
7. **مانگا**
8. **ئەنیمی**
9. **ستاف** (وەرگێڕ، دیزاینەر، تەکنیکار)
10. **ژێرنووسی ئینگلیزی**

## تایبەتمەندییە پێویستەکان:

### ١. سیستەمی زنجیرە فیلم:
- ئەگەر فیلمێک چەند بەشێکی هەبێت (وەک Avatar 1، 2، 3)، لە پەڕەیەکی تایبەتدا پێکەوە نیشانیان بدە
- هەموو بەشەکان بە ڕیزی کات پیشان بدە
- هەر بەشێک لینک بکە بۆ پەڕەی تایبەت بە خۆی

### ٢. سیستەمی گەڕانی پێشکەوتوو:
- گەڕانی گشتی بۆ: فیلم، زنجیرە، مانگا، ئەنیمی، ئەکتەر، ئەندامانی ستاف
- پێشنیاری گەڕانی ڕاستەوخۆ
- فلتەری گەڕان و تەواوکردنی خۆکار
- مێژووی گەڕان بۆ بەکارهێنەرە چووەتەژوورەوەکان

### ٣. سیستەمی فلتەرکردن:
فلتەرکردن بەپێی:
- **ژانەر**
- **ساڵ**
- **ڕیزکردن**: نوێترین، پڕبینراوترین، زۆرترین داهات، زۆرترین بودجە، ڕەیتینگ
- تێکەڵکردنی چەند فلتەرێک
- پاشەکەوتکردنی هەڵبژاردنی فلتەر

### ٤. پەڕەکانی ستاف:
- پەڕەی تایبەت بۆ وەرگێڕان، تەکنیکاران، دیزاینەران
- نیشاندانی ژمارەی ئەو فیلم/زنجیرانەی کاریان بۆ کردووە
- باجی جیا بۆ هەر جۆرە ڕۆڵێک
- پەڕەی پڕۆفایلی تایبەت بۆ هەر ئەندامێکی ستاف
- لیستی هەموو بەشداریەکانیان

### ٥. پەڕەی وردەکاری ناوەڕۆک (فیلم/زنجیرە):
دەبێت ئەمانە لەخۆبگرێت:
- **زمان**
- **جۆر** (فیلم/زنجیرە)
- **ژانەرەکان**
- **داهات**
- **بودجە**
- **ڕەیتینگ**
- **کات** (درێژی)
- **بەرواری دەرچوون**
- **کوالێتی** (HD، FullHD، 4K)
- **ئەکتەرەکان**
- **دەرهێنەر**
- **ڕێژەی بینەر** (ژمارەی سەردان)
- **وڵات**
- **وتەی بەناوبانگ** (دایەلۆگی ناسراو)
- **ئامانج** (مەبەستی فیلمەکە)
- **کورتە** (پوختەی چیرۆک)
- **تاگکردنی ستاف** (وەرگێڕ، دیزاینەر، تەکنیکارەکانی کارکردوون بۆی)

### ٦. سیستەمی پێداچوونەوە:
- زیادکردنی پێداچوونەوە لەگەڵ هەڵسەنگاندن
- جیاکردنەوەی **پێداچوونەوەی سپۆیلەر** و **پێداچوونەوەی ئاسایی**
- دەنگدان بۆ سوودمەندی پێداچوونەوەکان
- ڕیزکردنی پێداچوونەوەکان بەپێی بەروار، ڕەیتینگ، سوودمەندی

### ٧. سیستەمی پۆستی بەکارهێنەر:
- بەکارهێنەران دەتوانن پۆست دروستبکەن دەربارەی هەواڵی سینەما، زانیاریی فیلم
- پێداچوونەوەی ئەدمین پێش بڵاوکردنەوە
- سیستەمی حەزلێکردن/لێدوان لەسەر پۆستەکان
- تاگی هاوپۆل بۆ پۆستەکان

### ٨. پڕۆفایلی ئەکتەران:
- پەڕەی پڕۆفایلی تایبەت بۆ هەر ئەکتەرێک
- ژیانامە و زانیاریی کەسی
- لیستی تەواوی فیلمەکانیان
- لیستی ئەو فیلم/زنجیرانەی لەسەر پلاتفۆرمەکەدا بەردەستن

### ٩. سیستەمی چوونەژوورەوە:
- تۆمارکردن و چوونەژوورەوە
- پشتڕاستکردنەوەی ئیمەیڵ
- گەڕاندنەوەی تێپەڕەوشە
- چوونەژوورەوە بە سۆشیال میدیا

### ١٠. لیستەکان و پێشنیارەکانی بەکارهێنەر:
- **لیستی دڵخواز**
- **سەیرکردنی دواتر**
- **پێشنیارکراو بۆ خەڵکی تر**
- پڕۆفایلی گشتی کە دڵخواز و پێشنیارەکان نیشان دەدات
- سەیرکردنی دواتر تایبەتە (تەنها بەکارهێنەرەکە دەیبینێت)

### ١١. بەڕێوەبردنی پڕۆفایلی بەکارهێنەر:
بەکارهێنەر دەتوانێت دەستکاری بکات:
- وێنەی پڕۆفایل
- لینکی سۆشیال میدیا (تویتەر، ئینستاگرام، فەیسبووک، هتد)
- بەشی دەربارە/ژیانامە
- نیشاندانی بەشەکان: پۆستەکان، پێداچوونەوەکان، دڵخوازەکان، سەیرکردنی دواتر (تایبەت)، پێشنیارەکان

### ١٢. تەنها سێرڤەری دەرەکی بۆ ڤیدیۆ:
- **هیچ هۆستێکی ناوخۆیی بۆ ڤیدیۆ نەبێت**
- ئەدمین لینکی سێرڤەری دەرەکی زیاد دەکات (vidmoly، doodstream، هتد) لەکاتی پێشخستنی ناوەڕۆک
- تەنها iframe/embed ی ساکار بۆ پلەیەری دەرەکی
- **هیچ تایبەتمەندیەکی پلەیەری تایبەت نەبێت** (پلەی، پاوز، سیک، ڕیزیوم، ئەپلۆد)
- چەند هەڵبژاردەیەکی سێرڤەر بۆ هەر فیلم/ئەڵقەیەک

### ١٣. سیستەمی ڕاپۆرتکردن:
بەکارهێنەران دەتوانن کێشە ڕاپۆرت بکەن:
- **کێشەی سێرڤەر**
- **نزمی کوالێتی**
- **هەڵەی ژێرنووس**
- **کێشەی تایبەت** (بینوسێتەوە)
- ڕاپۆرتەکان دەچنە داشبۆردی ئەدمین
- ئەدمین دەتوانێت ڕاپۆرتەکان ببینێت و بەڕێوەیان ببات

### ١٤. سێرڤەری خاوێنکردنەوەی وێنە:
- سێرڤەرێکی جیای Express.js بۆ وێنەکان
- Sharp.js بۆ قەبارە گۆڕین و خاوێنکردنەوەی داینامیک
- کاشکردنی Redis بۆ وێنە پرۆسێسکراوەکان
- پێشکەشکردنی وێنە بەپێی پێداویستی ئامێر (مۆبایل، تابلێت، دێسکتۆپ)
- پشتگیری فۆرماتی WebP لەگەڵ فۆڵباک

## پێداویستییەکانی داشبۆردی ئەدمین:
پانێڵی بەڕێوەبردنی فراوان لەگەڵ:
- بەڕێوەبردنی ناوەڕۆک (زیادکردن/دەستکاریکردن/سڕینەوەی فیلم، زنجیرە، مانگا، ئەنیمی)
- بەڕێوەبردنی بەکارهێنەران
- مۆدێریشنی پێداچوونەوەکان
- مۆدێریشنی پۆستەکان
- بەڕێوەبردنی ڕاپۆرتەکان
- بەڕێوەبردنی ستاف
- بەڕێوەبردنی داتابەیسی ئەکتەر/دەرهێنەر
- ئاماری و ستاتیستیک
- بەڕێوەبردنی لینکی سێرڤەر بۆ هەر ناوەڕۆکێک
- کردارە کۆمەڵەییەکان
- خشتەکردنی ناوەڕۆک
- ڕێکخستنی SEO بۆ هەر ناوەڕۆکێک
- بەڕێوەبردنی کاش

- بەڕێوەبردنی تریلەر

- دیزاینی ڕێسپۆنسیڤ بۆ هەموو ئامێرەکان
- پشتگیری PWA
- تایبەتمەندیی هاوبەشکردنی سۆشیاڵ

## پێداویستییەکانی پێکهاتەی پڕۆژە:
1. کەمتر لە 100 فایل نەبێت
2. پێکهاتەی فۆڵدەری ڕێکخراو
3. فایلە جیاکانی ڕێکخستن
4. گۆڕاوەکانی ژینگە بە باشی بەڕێوە ببرێن
5. دۆکیومێنتکردنی فراوان
6. دۆکیومێنتکردنی API
7. دۆکیومێنتکردنی سکیمای داتابەیس
8. ڕێنماییەکانی دیپلۆیکردن

## دەرئەنجامەکان:
1. **یەکەم**: پێکهاتەی تەواوی پڕۆژە بنێرە لەگەڵ هەموو ناوی فۆڵدەر و فایلەکان
2. **دواتر**: چاوەڕێی پشتڕاستکردنەوە بکە پێش بەردەوامبوون
3. **لەبیربە**: هیچ سێرڤەر/پلەیەری ناوخۆیی بۆ ڤیدیۆ نەبێت - تەنها لینکی دەرەکی

پێکهاتەی تەواوی پڕۆژە دروست بکە سەرەتا، هەموو فۆڵدەر و فایلەکان (کەمتر لە 100+ فایل نەبێت) نیشان بدە، بە شێوەیەکی پڕۆفیشناڵ ڕێکخراو بێت و ناوەکانیان بە ئینگلیزی بێت.
